# X Clone

A responsive clone of **X (formerly Twitter)** built to recreate its modern user interface and improve understanding of front-end development concepts such as layouts, responsiveness, and utility-first styling.

## Features

* Responsive X-inspired interface
* Navigation sidebar
* Feed section
* Search and trends section
* "Who to Follow" recommendations
* Hover animations and transitions
* Clean and modern design using Tailwind CSS

## Tech Stack

* HTML5
* CSS3
* Tailwind CSS

## Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/x-clone.git
cd x-clone
```

### 2. Install dependencies

```bash
npm install
```

### 3. Start Tailwind in watch mode

```bash
npm run dev
```

### 4. Open the project

Open `X.html` in your browser or use a live server extension.
